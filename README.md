# iterator
Создаёт коллекцию MyCollection с методами добавления и получения элементов.

Итератор MyIterator перебирает элементы коллекции последовательно.

В Main создаём коллекцию, добавляем элементы, потом через итератор выводим их на экран по очереди.

Паттерн Iterator позволяет обходить элементы коллекции без раскрытия её внутренней структуры.
